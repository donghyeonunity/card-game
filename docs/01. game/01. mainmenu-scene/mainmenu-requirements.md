> **문서 버전**: v1.0

> **관련 문서**: [PRD.md](../../00.%20project/PRD.md)

## 1. 개요

### 1.1 씬 정보
| 항목 | 내용 |
|------|------|
| **씬 이름** | MainMenu |
| **목적** | 게임의 진입점. 플레이어가 새 게임을 시작하거나, 저장된 게임을 이어하거나, 설정을 변경할 수 있는 화면 |
| **진입 조건** | 게임 실행 시 최초 진입, 또는 게임 내에서 메인메뉴로 복귀 시 |

### 1.2 핵심 기능 요약
- 새 게임 시작 (캐릭터 선택 씬으로 이동)
- 저장된 런 이어하기
- 게임 설정 변경
- 게임 종료

## 3. UI 요소 상세

### 3.1 타이틀 로고

| 속성 | 값 |
|------|------|
| **타입** | 이미지 (Image) |
| **위치** | 화면 중앙, 상단~중간 영역 |
| **크기** | 화면 너비의 40~50% (반응형) |
| **애니메이션** | 선택사항: 미세한 호흡 효과 (scale pulse) |

### 3.2 메뉴 버튼

#### 3.2.1 공통 속성

| 속성 | 값 |
|------|------|
| **배치** | 좌측 하단, 세로 정렬 |
| **버튼 간격** | 일정한 간격 유지 |
| **정렬** | 좌측 정렬 |

#### 3.2.2 버튼 목록

| 버튼명 | ID | 표시 조건 | 동작 |
|--------|-----|-----------|------|
| 게임 시작하기 | btn_new_game | 항상 표시 | 캐릭터 선택 씬으로 이동 |
| 계속하기 | btn_continue | 저장된 런이 있을 때만 표시 | 저장된 게임 로드 후 게임 씬으로 이동 |
| 설정 | btn_settings | 항상 표시 | 설정 팝업 열기 |
| 종료 | btn_quit | 항상 표시 | 게임 종료 |

#### 3.2.3 버튼 상태

| 상태 | 시각적 변화 | 사운드 |
|------|-------------|--------|
| **Normal** | 기본 상태 | - |
| **Hover** | 확대/강조 애니메이션 | 호버 효과음 |
| **Pressed** | 눌림 효과 | 클릭 효과음 |
| **Disabled** | 사용 안 함 (해당 버튼 없음) | - |

### 3.3 버전 정보

| 속성 | 값 |
|------|------|
| **타입** | 텍스트 (Text) |
| **위치** | 화면 우측 하단 |
| **포맷** | "v{Major}.{Minor}.{Patch}" (예: v1.0.0) |
| **스타일** | 작은 폰트, 낮은 불투명도 |

## 4. 기능 명세

### 4.1 게임 시작하기

| 항목 | 내용 |
|------|------|
| **트리거** | "게임 시작하기" 버튼 클릭 |
| **동작** | 캐릭터 선택 씬으로 전환 |
| **전환 효과** | 페이드 아웃 → 씬 로드 → 페이드 인 |

### 4.2 계속하기

| 항목 | 내용 |
|------|------|
| **표시 조건** | 저장된 런 데이터가 존재할 때만 표시 |
| **트리거** | "계속하기" 버튼 클릭 |
| **동작** | 저장된 런 데이터 로드 → 게임 씬으로 전환 |
| **전환 효과** | 페이드 아웃 → 데이터 로드 → 씬 로드 → 페이드 인 |

#### 저장 시스템 참고
- 런 중간 저장 지원 (플레이 중 저장하고 나가기 가능)
- 저장 데이터가 없으면 버튼 자체를 숨김 (비활성화가 아님)

### 4.3 설정

| 항목 | 내용 |
|------|------|
| **트리거** | "설정" 버튼 클릭 |
| **동작** | 설정 팝업/패널 열기 |
| **팝업 타입** | 오버레이 팝업 (씬 전환 없음) |

#### 설정 항목

| 카테고리 | 설정 항목 | 타입 | 설명 |
|----------|----------|------|------|
| **사운드** | BGM 볼륨 | 슬라이더 (0~100) | 배경 음악 볼륨 조절 |
| **사운드** | 효과음 볼륨 | 슬라이더 (0~100) | 효과음 볼륨 조절 |
| **화면** | 해상도 | 드롭다운 | 화면 해상도 선택 |
| **화면** | 전체화면 | 토글 | 전체화면/창 모드 전환 |
| **언어** | 언어 선택 | 드롭다운 | 게임 언어 변경 |

### 4.4 종료

| 항목 | 내용 |
|------|------|
| **트리거** | "종료" 버튼 클릭 |
| **동작** | 게임 애플리케이션 종료 |
| **확인 절차** | 선택사항: 종료 확인 팝업 표시 |

## 6. 애니메이션/효과

### 6.1 배경 애니메이션

| 항목 | 내용 |
|------|------|
| **타입** | 파티클, 빛 효과, 움직이는 요소 등 |
| **목적** | 화면에 생동감 부여, 게임 분위기 조성 |
| **성능** | 저사양 PC에서도 원활하게 동작해야 함 |

### 6.2 버튼 애니메이션

| 상태 | 애니메이션 |
|------|------------|
| **Hover** | 스케일 확대 (1.0 → 1.05~1.1) 또는 글로우 효과 |
| **Click** | 스케일 축소 후 복귀 (눌림 효과) |
| **전환** | 부드러운 이징 (ease-out) |

### 6.3 씬 전환 효과

| 효과 | 설명 |
|------|------|
| **페이드 아웃** | 현재 화면이 서서히 어두워짐 |
| **페이드 인** | 새 화면이 서서히 밝아짐 |
| **전환 시간** | 0.3~0.5초 권장 |

## 8. 기술 요구사항

### 8.1 저장 데이터 확인
- 게임 시작 시 저장 데이터 존재 여부 확인
- 저장 데이터 유효성 검증
- 결과에 따라 "계속하기" 버튼 표시/숨김

### 8.2 설정 데이터 관리
- 설정 값 로컬 저장 (PlayerPrefs 또는 파일)
- 게임 시작 시 설정 값 로드 및 적용
- 설정 변경 시 즉시 적용

### 8.3 반응형 UI
- 다양한 해상도 지원 (16:9, 16:10, 21:9 등)
- UI 요소 앵커 및 스케일링 적용
