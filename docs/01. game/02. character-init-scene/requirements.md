# 캐릭터 초기화 씬 요구사항 문서

## 1. 씬 개요

### 1.1 목적
- 매 런(run) 시작 시 플레이어의 캐릭터와 특성을 선택하는 씬
- 이번 런의 **초기 방향성**을 결정하는 Layer 2 중간 판단
- 선택한 조합이 배틀씬의 즉각 반응 판단에 영향을 미침

### 1.2 진입 조건
- **이전 씬**: 메인메뉴 씬 (01. mainmenu-scene)
- **진입 트리거**: "새 게임 시작" 또는 "다시 도전" 버튼 클릭
- **빈도**: 매 런마다 1회 (런이 끝나면 메인메뉴로 돌아감)

### 1.3 다음 씬
- **첫 번째 배틀씬** 또는 **이벤트 씬** (던전 구조에 따라)
- 선택 완료 후 즉시 게임 시작

### 1.4 핵심 경험
> "이번 런에서는 어떤 전략으로 갈까?"

**플레이어 예상 감정:**
- "지난번에는 무모한 전사가 재밌었는데, 이번엔 신중한 법사를 해볼까?"
- "포인트를 어떻게 분배할지 고민되네... 몰빵? 아니면 분산?"
- "이 조합이 배틀에서 어떻게 느껴질지 기대돼!"

## 2. 선택 포인트

### 2.1 캐릭터 선택 (3종 중 1개)

| 캐릭터 | 기본 스탯 | 특징 | 전용 카드 특화 |
|--------|-----------|------|----------------|
| **전사** | HP: 60<br>에너지: 3 | 높은 체력, 공격 중심 | 공격/방어 카드 높은 출현율 |
| **법사** | HP: 40<br>에너지: 4 | 낮은 체력, 에너지 여유 | 주문/드로우 카드 높은 출현율 |
| **도적** | HP: 50<br>에너지: 3 | 밸런스형, 속임수 중심 | 회피/에너지 조작 카드 높은 출현율 |

**플레이어 예상 감정:**
- 전사: "체력이 많으니까 안정적이겠어, 실수해도 버틸 수 있을 것 같아" (안정성)
- 법사: "에너지가 많으니까 선택지가 넓겠는데, HP가 낮아서 긴장되네" (변동성)
- 도적: "무난한 선택인데, 특색이 약한가?" (효율성)

---

### 2.2 특성 선택 (포인트 분배)

**총 포인트: 5점**

5개 특성에 자유롭게 분배 가능:

| 특성 | 제공 카드 (1점) | 강화 카드 (3점) | 최강 카드 (5점) | 패시브 효과 (5점) |
|------|----------------|----------------|----------------|-------------------|
| **신중함** | "방어 태세"<br>(방어 +5) | "철벽 수비"<br>(방어 +10, 에너지 회복 +1) | "불굴의 의지"<br>(방어 +15, 에너지 회복 +2) | 모든 방어 카드 효과 +20% |
| **무모함** | "무모한 돌격"<br>(공격 +8, HP -3) | "광란의 일격"<br>(공격 +15, HP -5, 적 기절 20%) | "죽음의 춤"<br>(공격 +25, HP -10, 적 기절 50%) | HP 50% 이하일 때 공격력 +30% |
| **분석적** | "전략 분석"<br>(드로우 +1) | "심층 분석"<br>(드로우 +2, 에너지 -1) | "완벽한 계획"<br>(드로우 +3, 손패 상한 +2) | 카드 드로우 시 에너지 +1 (턴당 1회) |
| **직감적** | "섬광 같은 직감"<br>(랜덤 카드 1장 즉시 드로우) | "운명의 속삭임"<br>(랜덤 카드 2장, 1장 선택) | "완벽한 타이밍"<br>(덱에서 원하는 카드 1장 서치) | 카드 비용 -1 (턴당 1회) |
| **협상가** | "거래의 달인"<br>(골드 +20) | "유리한 계약"<br>(골드 +40, 상점 가격 -10%) | "대상인의 수완"<br>(골드 +70, 상점 가격 -20%, 보상 선택지 +1) | 배틀 승리 시 골드 +50% |

**플레이어 예상 감정:**

**1점 × 5개 (분산형):**
- "모든 특성을 조금씩 맛볼 수 있어! 다양한 상황에 대응 가능하겠어"
- **가치**: 확장성 (다양한 카드 확보)
- **트레이드오프**: 카드 하나하나의 위력이 약함

**3점 + 1점 + 1점 (밸런스형):**
- "한 개는 확실히 강하게 가고, 나머지는 보조로... 안정적인 것 같아"
- **가치**: 효율성 + 일부 확장성
- **트레이드오프**: 중간이라서 확실한 방향성이 약할 수도

**5점 몰빵 (집중형):**
- "이 카드 하나로 승부를 보겠어! 엄청 강력하겠는데?"
- "하지만 이 카드에 의존하게 되면... 못 뽑으면 위험할 것 같아"
- **가치**: 효율성 극대화 + 패시브 효과
- **트레이드오프**: 다양성 포기, 리스크 높음

### 2.3 조합 보너스 (시너지 카드)

특정 특성 조합 시 추가 시너지 카드 제공:

| 조합 | 조건 | 시너지 카드 | 효과 |
|------|------|------------|------|
| **전술가** | 신중함 2점 + 분석적 2점 | "전술적 후퇴" | 방어 +5, 드로우 +1, 에너지 회복 +1 |
| **광전사** | 무모함 3점 + 직감적 2점 | "광란의 직감" | 공격 +12, HP -5, 다음 카드 비용 -1 |
| **상인 전략가** | 협상가 2점 + 분석적 2점 | "계산된 투자" | 골드 30 소모 → 드로우 +3, 에너지 +1 |
| **위험한 도박** | 무모함 2점 + 협상가 2점 | "위험한 거래" | HP -10, 골드 +50, 랜덤 레어 카드 1장 |
| **신중한 투자자** | 신중함 2점 + 협상가 2점 | "안전한 수익" | 방어 +3, 골드 +15, 상점 재입고 1회 |

**플레이어 예상 감정:**
- "아, 이 조합을 하면 보너스 카드를 받는구나!"
- "5점 몰빵도 좋지만, 2+2+1로 조합 보너스를 노려볼까?" (가치 충돌!)
- "이 시너지 카드가 내 전략에 딱 맞을 것 같아" (발견의 기쁨)

## 3. 포인트 시스템 상세

### 3.1 임계값 시스템

**포인트별 효과:**

```
0점 → 효과 없음
1점 → 기본 카드 1장
2점 → 기본 카드 1장 (변화 없음, 조합 보너스 조건용)
3점 → 강화 카드 1장 (기본 카드 대체)
4점 → 강화 카드 1장 (변화 없음)
5점 → 최강 카드 1장 + 패시브 효과
```

**핵심 설계 의도:**
- 1점, 3점, 5점이 의미 있는 투자
- 2점, 4점은 "조합 보너스 조건 달성"을 위한 선택지
- 예: "신중함 2점 + 분석적 2점 + 무모함 1점" → 전술가 시너지 획득

**플레이어 예상 감정:**
- "2점 찍는 건 손해인 것 같은데... 조합 보너스를 노린다면 의미 있네!"
- "3점이면 카드가 확 강해지네, 여기까지는 찍어야겠어"
- "5점 몰빵하면 패시브까지 생기는구나! 완전 특화 빌드네"

---

### 3.2 분배 예시

**총 5점 분배 시나리오:**

| 시나리오 | 분배 | 획득 카드 | 전략 의도 | 플레이어 감정 |
|----------|------|-----------|----------|---------------|
| **몰빵형** | 신중함 5점 | "불굴의 의지" + 방어 패시브 | 방어 특화 빌드 | "방어만큼은 최강! 버티면서 싸우자" |
| **2강화형** | 신중함 3점<br>분석적 2점 | "철벽 수비" + "전략 분석"<br>+ "전술적 후퇴" (시너지) | 방어 + 드로우 밸런스 | "방어도 강하고, 카드도 많이 뽑고, 시너지까지!" |
| **분산형** | 신중함 1점<br>무모함 1점<br>분석적 1점<br>직감적 1점<br>협상가 1점 | 모든 기본 카드 5장 | 만능형, 상황 대응 | "어떤 상황에도 대응할 수 있어, 유연하게!" |
| **조합 사냥형** | 무모함 3점<br>직감적 2점 | "광란의 일격" + "운명의 속삭임"<br>+ "광란의 직감" (시너지) | 공격 + 운빨 빌드 | "한 방이 쎄고, 운도 따라주면 최강!" |

## 4. 카드 획득 시스템

### 4.1 전용 카드 풀 + 가중치 시스템

**구조:**

```
전체 카드 풀 = 전용 카드 풀 + 공통 카드 풀

- 전사 전용 카드: 30장 (공격/방어 특화)
- 마법사 전용 카드: 30장 (주문/드로우 특화)
- 도적 전용 카드: 30장 (회피/에너지 조작 특화)
- 공통 카드: 10장 (모든 캐릭터 사용 가능)
```

**배틀 보상 카드 출현 확률:**

| 카드 유형 | 출현 확률 |
|-----------|----------|
| 자신의 전용 카드 | 70% |
| 공통 카드 | 25% |
| 다른 캐릭터 전용 카드 | 5% (희귀) |

**플레이어 예상 감정:**
- "보상으로 주로 내 전용 카드가 나오네, 빌드를 발전시키기 좋아"
- "어? 마법사 카드가 떴네! 희귀하긴 한데, 전사랑 시너지가 있을까?" (발견의 놀라움)
- "공통 카드는 어떤 캐릭터든 쓸 수 있으니까 안정적이야"

---

### 4.2 캐릭터 × 특성 독립성

**핵심 원칙:**
- 캐릭터 선택은 **획득 가능한 카드 풀**에 영향
- 특성 선택은 **시작 덱에 추가되는 카드**에 영향
- 두 시스템은 **독립적으로 작동**

**예시:**
```
전사 + 신중함 3점 선택 시:

1. 캐릭터 효과 (전사)
   - 기본 스탯: HP 60, 에너지 3
   - 배틀 보상에서 전사 전용 카드 70% 출현

2. 특성 효과 (신중함 3점)
   - 시작 덱에 "철벽 수비" 카드 추가
   - 다른 캐릭터가 선택해도 같은 카드 제공

3. 독립성의 의미
   - 마법사 + 신중함 3점 선택 시
   - 같은 "철벽 수비" 카드 획득
   - 하지만 마법사는 HP 40이라서 방어 카드의 체감이 다름
```

**플레이어 예상 감정:**
- "어떤 캐릭터를 골라도 신중함은 같은 카드를 주는구나, 명확해"
- "하지만 HP가 낮은 마법사는 방어 카드가 더 중요하게 느껴질 것 같아"
- "캐릭터랑 특성을 어떻게 조합하느냐에 따라 체감이 달라지겠네"

### 5.2 인터랙션 요구사항

**5.2.1 캐릭터 선택**
- 3개 캐릭터 중 1개 라디오 버튼 선택
- 선택 시 테두리 강조, 스탯 정보 표시
- 호버 시 툴팁으로 상세 설명 표시

**5.2.2 특성 포인트 분배**
- 각 특성별 0~5점 슬라이더 또는 +/- 버튼
- 실시간으로 획득 카드 미리보기 표시
- 1점, 3점, 5점 임계값에서 시각적 피드백 (카드 이미지 변경)
- 총 포인트 초과 시 경고 메시지

**5.2.3 조합 보너스 표시**
- 조건 달성 시 ✓ 표시와 함께 시너지 카드 표시
- 조건 미달 시 회색으로 표시 (조건 명시)

**5.2.4 최종 확인**
- "최종 시작 덱 미리보기" 패널에서 모든 획득 카드 나열
- "카드 상세 보기" 버튼 → 각 카드 효과 상세 설명 모달
- "게임 시작" 버튼 → 다음 씬으로 전환

### 5.3 플레이어 피드백

**실시간 피드백:**
- 포인트 분배 시 즉시 획득 카드 이미지 업데이트

**플레이어 예상 감정:**

## 7. PRD 연결

### 7.1 Layer 2 중간 판단

**이 씬의 위치:**
- **판단 계층**: Layer 2 (Reflective Mid Judgment)
- **빈도**: 매 런마다 1회 (간헐적)
- **무게**: 중간 (런 전체에 영향)

**PRD 핵심 가치 반영:**
> "여러 반응 판단의 결과가 반영된다"
- 이전 런에서의 경험이 이번 선택에 영향을 줌
- "지난번에 신중함이 좋았으니 이번에도..." 또는 "이번엔 다르게 해보자"

> "복수의 가치 축이 동시에 명시된다"
- 포인트 분배 시 **효율 ↔ 확장** 충돌 (몰빵 vs 분산)
- 조합 보너스 시 **통제 ↔ 자유** 충돌 (계산된 조합 vs 자유로운 분배)

---

### 7.2 가치 충돌 매핑

| 선택 상황 | 가치 축 충돌 | 플레이어 감정 |
|-----------|-------------|---------------|
| **5점 몰빵 vs 1점 분산** | 효율 ↔ 확장 | "강한 한 방 vs 다양한 대응력" |
| **3점 강화 카드 vs 조합 보너스** | 통제 ↔ 자유 | "확실한 카드 vs 시너지 도박" |
| **캐릭터 선택** | 안정 ↔ 변동 | "전사의 안정성 vs 마법사의 위험성" |
| **무모함 특성** | 단기 ↔ 장기 | "초반 강력함 vs HP 손실 누적" |

---

### 7.3 배틀씬 연결

**선택 → 반응 판단 영향:**

| 이 씬 선택 | 배틀씬 영향 | 플레이어 감정 변화 |
|-----------|------------|-------------------|
| 신중함 5점 | 방어 카드 강력, 방어 패시브 | "실수해도 버틸 수 있어" → 안정감 |
| 무모함 5점 | 공격 카드 강력, HP 소모 큼 | "한 방이 쎈데 HP가 빠르게 줄어" → 긴장감 |
| 분석적 3점 | 드로우 많음, 에너지 부족 | "카드는 많은데 다 못 써" → 선택의 고통 |
| 직감적 5점 | 원하는 카드 서치 가능 | "필요한 카드를 골라 쓸 수 있어" → 통제감 |
| 협상가 5점 | 골드 많음, 상점 활용 중요 | "배틀 외적 성장에 투자" → 장기 전략 |

**핵심:**
- 이 씬의 선택이 배틀씬의 **즉각 반응 판단 지형**을 바꿈
- PRD의 "중간 판단이 반응 판단을 다시 긴장 상태로 되돌리는 장치"에 부합

## 9. 향후 논의 사항

### 9.1 밸런스 조정 대상

**테스트 후 조정 가능:**
- 총 포인트 (현재 5점 → 7점 또는 10점으로 증가?)
- 임계값 (1, 3, 5 → 1, 2, 4로 변경?)
- 조합 보너스 조건 (너무 까다롭거나 쉬운지)
- 패시브 효과 강도 (5점 몰빵이 너무 강하거나 약한지)

---

### 9.2 확장 가능성

**향후 추가 가능한 요소:**

1. **캐릭터 언락 시스템**
   - 처음엔 전사만, 이후 마법사/도적 해금
   - 플레이어 감정: "새 캐릭터를 해금했다!" (성취감)

2. **특성 언락 시스템**
   - 특정 조건 달성 시 새로운 특성 추가
   - 플레이어 감정: "이 특성은 어떤 효과일까?" (발견의 기대)

3. **시작 포인트 증가 시스템**
   - 업적 달성 시 시작 포인트 +1 (최대 10점)
   - 플레이어 감정: "더 자유롭게 조합을 만들 수 있어!" (성장감)

4. **캐릭터 × 특성 시너지 추가**
   - 현재는 독립적이지만, 특정 조합에 보너스
   - 예: "전사 + 무모함" → 추가 공격력 +2
   - 플레이어 감정: "이 조합이 특별하구나!" (발견의 재미)

5. **히든 조합 보너스**
   - UI에 표시되지 않는 숨겨진 조합
   - 플레이어 감정: "어? 이런 조합이 있었어?" (서프라이즈)

---

### 9.3 UI/UX 개선 후보

**테스트 후 검토:**
- 카드 미리보기 이미지 추가 (현재는 텍스트만)
- 특성 선택 튜토리얼 (처음 플레이 시)
- "추천 조합" 버튼 (초보자용)
- "랜덤 선택" 버튼 (빠른 플레이용)
- 이전 런 기록 표시 ("지난번: 전사 + 신중함 3점")

---

### 9.4 데이터 분석 항목

**추후 수집할 데이터:**
- 가장 많이 선택되는 캐릭터 (밸런스 확인)
- 가장 많이 선택되는 특성 (인기도)
- 포인트 분배 패턴 (몰빵 vs 분산 비율)
- 조합 보너스 달성률 (너무 어렵거나 쉬운지)
- 승률과 선택의 상관관계 (어떤 조합이 강한지)
