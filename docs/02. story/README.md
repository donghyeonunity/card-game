# 스토리 & 세계관

> **문서 목적**: 게임의 스토리, 세계관, 캐릭터 설정 정리
> **스토리 철학**: "판타지 세계 + 현실 고민 함축" - 표면은 판타지, 내면은 공감

---

## 핵심 구조

| 요소 | 설명 |
|------|------|
| **던전** | 미스터리한 존재. 최상부에 도달한 자 없음 |
| **플레이어** | 각자의 사연을 가지고 혼자 던전에 진입 |
| **목표** | 던전 최상부 도달 → 각자의 소원 성취 |
