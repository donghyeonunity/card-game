# 캐릭터 선택 씬 와이어프레임 v4.0

## 개요

- **씬 목적**: 매 런 시작 시 캐릭터와 특성을 선택
- **기준 해상도**: 1920x1080
- **레이아웃**: 펼쳐진 고서(책) 형태의 좌우 페이지
- **스타일**: 다크 판타지, 세피아톤
- **상세 스펙**: `wireframe-figma.json` 참조

### 영역 구성

| 영역 | 크기 | 설명 |
|------|------|------|
| **SCREEN** | 1920x1080 | 전체 화면 |
| **BOOK_FRAME** | 1760x880 | 고서 프레임 (세피아 톤 배경) |
| **LEFT_PAGE** | ~840px | 캐릭터 일러스트 + 대사 박스 |
| **RIGHT_PAGE** | ~840px | 캐릭터 정보 + 특성 선택 + 카드 미리보기 |
| **FOOTER** | 1760x80 | 뒤로/캐릭터탭/게임시작 버튼 |

## 컴포넌트 요약

### LEFT_PAGE (좌측 페이지)

| 컴포넌트 | 설명 |
|----------|------|
| `character_portrait` | 캐릭터 일러스트 (세피아톤/빈티지 스타일, 상반신) |
| `dialogue_box` | 캐릭터 대사 박스 (반투명 배경) |

### RIGHT_PAGE (우측 페이지)

| 컴포넌트 | 설명 |
|----------|------|
| `page_title` | "캐릭터 선택" 타이틀 |
| `character_info` | 캐릭터 이름, 체력, 에너지, 특화 |
| `traits_section` | 특성 버튼 (3x2 그리드) + 설명 박스 |
| `cards_section` | 획득 카드 4장 미리보기 |

### FOOTER (하단 네비게이션)

| 컴포넌트 | 설명 |
|----------|------|
| `back_button` | "← 뒤로" 버튼 (메인메뉴 이동) |
| `character_tabs` | 캐릭터 탭 버튼 3개 (전사/법사/도적) |
| `start_button` | "게임 시작 →" 버튼 |

### OVERLAY (팝업)

| 컴포넌트 | 설명 |
|----------|------|
| `tooltip` | 특성/카드 호버 시 상세 정보 |
| `confirm_dialog` | 뒤로가기 확인 다이얼로그 |

## 데이터

### 캐릭터

| 캐릭터 | HP | 에너지 | 특화 | 대사 |
|--------|-----|--------|------|------|
| **전사** | 60 | 3 | 공격/방어 | "어머니 조금만... 조금만 더 버텨주세요" |
| **법사** | 40 | 4 | 주문/드로우 | "금지된 지식의 끝에 진리가 있을까?" |
| **도적** | 50 | 3 | 회피/에너지 | "반드시 찾아내겠어... 살아서 돌아와야 해" |

### 특성 (5~6개, 3x2 그리드)

| 특성 | 계열 | 효과 (예시) |
|------|------|-------------|
| 신중함 | 방어 | 받는 피해 -10% |
| 무모함 | 공격 | 공격력 +15%, 피해 +10% |
| 분석적 | 드로우 | 턴 시작 시 추가 드로우 |
| 직감적 | 서치 | 특정 카드 서치 가능 |
| 협상가 | 골드 | 획득 골드 +20% |
| (TBD) | - | 추가 특성 슬롯 |

## 인터랙션

| 요소 | 동작 |
|------|------|
| **캐릭터 탭** | 클릭 → 캐릭터 전환 (일러스트/정보/대사 갱신) |
| **특성 버튼** | 클릭 → 선택/해제 토글, 설명 박스 갱신 |
| **카드 미리보기** | 호버 → 카드 상세 툴팁 |
| **게임 시작** | 클릭 → 배틀씬으로 이동 |
| **뒤로** | 클릭 → 확인 다이얼로그 → 메인메뉴 |
