## 다음 작업 (우선순위순)

### 1. 몬스터 상세 설계

| 작업 | 설명 | 우선순위 |
|------|------|----------|
| **보스 컨셉** | 각 구역 보스 구체적 설계 (외형, 능력, 배경) | 높음 |
| **몬스터 종족** | 어떤 종족들이 있는지, 각 종족 특성 | 높음 |
| **일반/엘리트 몬스터** | 각 종족별 일반몹, 엘리트 디자인 | 중간 |
| **종족 간 관계** | 동맹? 적대? 무관심? | 낮음 |

### 2. 세계관 확장

| 작업 | 설명 | 우선순위 |
|------|------|----------|
| **전체 세계관 구조** | 던전 외부 세계는 어떤 곳인가 | 중간 |
| **던전의 역사** | 던전은 언제부터 있었나, 왜 생겼나 | 중간 |
| **던전 최상부 정체** | 끝에 무엇이 있는가 (핵심 미스터리) | 높음 |

### 3. 캐릭터 상세

| 작업 | 설명 | 우선순위 |
|------|------|----------|
| **법사 배경** | 왜 추방되었나, 무슨 지식을 찾나 | 중간 |
| **전사 배경** | 어떤 전쟁이었나, 어머니 병의 원인 | 중간 |
| **도적 배경** | 전쟁 후 왜 도적이 되었나 | 낮음 |

### 4. 스토리 구조

| 작업 | 설명 | 우선순위 |
|------|------|----------|
| **중간 반전 포인트** | 던전 중반에서 발견하는 충격적 진실 | 높음 |
| **엔딩 분기** | 캐릭터별 엔딩, 선택에 따른 결말 | 낮음 |
| **캐릭터 연결 이벤트** | 도적이 전사 흔적 발견하는 장면 등 | 낮음 |

## 추천 진행 순서

```
1단계: 핵심 미스터리 확정
   └── 던전 최상부에 무엇이 있는가?
   └── 중간 반전 포인트는 무엇인가?
         │
         ▼
2단계: 보스/종족 설계
   └── 보스 컨셉 (테마, 외형, 배경)
   └── 각 구역 몬스터 종족
         │
         ▼
3단계: 세계관 확장
   └── 던전 외부 세계
   └── 던전의 역사
         │
         ▼
4단계: 캐릭터 상세
   └── 각 캐릭터 세부 배경
   └── 엔딩 분기
```
