# 리치 보스 비주얼 디자인 가이드

## 1. 개요

### 캐릭터 정보
| 항목 | 내용 |
|------|------|
| **이름** | 리치 (The Lich) |
| **역할** | 1챕터 보스 |
| **정체** | 천 년 전 영생 의식을 주도한 궁정 마법사 |
| **목적** | 죽은 모험가들을 영원한 군대로 만듦 |
| **능력** | 시체 부활, 영혼 속박 |

### 디자인 키워드
- **타락한 왕족**: 과거의 권력과 현재의 부패
- **영혼 착취자**: 플레이어(구원자)와 대비되는 악역
- **고대의 존재**: 천 년의 세월이 느껴지는 고풍스러움

---

## 2. 레퍼런스 분석

### Slay the Spire
- 픽셀 캐릭터와 일러스트 카드의 조화
- **명확한 실루엣** 중시 (전투 중 가독성)
- 보스는 화면의 많은 공간 차지 (위압감)
- 제한된 팔레트로 강렬한 인상

### Darkest Dungeon
- Necromancer 보스: 고딕 다크 판타지 스타일
- 무거운 라인, 음울한 색감
- 언데드 소환 매커니즘 (리치와 유사)
- 캐릭터의 비극적 배경 강조

### 픽셀 아트 보스 원칙
- 제한된 해상도에서 강렬한 인상
- 명확한 시각 언어 (왕관 = 지도자, 해골 = 언데드)
- 애니메이션을 통한 생동감 부여
- 이펙트로 능력과 상태 전달

## 3. 캐릭터 컨셉

### 실루엣 디자인
```
        ◆◆◆ (왕관 - 기울어지고 일부 부서짐)
         ▼
      ╭─────╮
      │ ◉ ◉ │ (해골 얼굴, 보라색 빛나는 눈)
      ╰──┬──╯
    ╭────┴────╮
   ╱           ╲
  │   ~~~~~~   │ (로브 - 찢어진 부분에서 에너지 누출)
  │   ~~~~~~   │
  │     │      │
  ╰─────┴─────╯
        ║
       ◇◇◇ (지팡이 - 보라색 수정)
```

### 핵심 디자인 요소

#### 왕관
- **형태**: 전통적인 왕관이지만 일부 부서지고 기울어짐
- **의미**: 과거의 영광과 현재의 타락을 동시에 표현
- **색상**: 부패한 금색 (`#9D8B4F`, `#6B5D3A`)
- **디테일**: 보석 자리가 비어있거나 금이 간 모습

#### 해골 얼굴
- **형태**: 인간 두개골 기반, 턱뼈 연결
- **눈**: 빈 눈구멍에서 보라색 에너지 발광
- **표현**: 위엄 있지만 섬뜩한 느낌
- **색상**: 회색-보라 계열 (`#4A3B52` → `#C4B8CC`)

#### 로브
- **형태**: 화려했던 궁정 마법사 로브의 부패한 버전
- **상태**: 곳곳이 찢어지고 닳음
- **특징**: 찢어진 틈으로 보라색 영혼 에너지 누출
- **색상**: 진한 보라 계열 (`#3B1F47` → `#9D2AA5`)

#### 지팡이
- **형태**: 긴 마법 지팡이, 상단에 큰 수정
- **수정**: 보라색, 내부에 영혼들이 소용돌이침
- **역할**: 영혼 속박 마법의 매개체
- **애니메이션**: 수정이 지속적으로 맥동

- **금색**: 과거의 영광, 현재는 녹슬고 퇴색

## 5. 스프라이트 사양

### 기술 규격
| 항목 | 값 | 비고 |
|------|---|------|
| **해상도** | 96 x 128 픽셀 | 일반 적(48x64)의 2배 |

### 레이어 구성

2. **그림자 레이어**: 발 아래 그림자
3. **캐릭터 레이어**: 본체 스프라이트
4. **이펙트 레이어**: 눈 발광, 에너지 누출

## 6. 애니메이션 키프레임

### Idle (대기)
| 항목 | 값 |
|------|---|
| **프레임 수** | 6 |
| **FPS** | 6 (느린 움직임) |
| **루프** | O |

**동작 설명**:
- 프레임 1-2: 로브가 왼쪽으로 살짝 흔들림
- 프레임 3-4: 중앙 위치, 수정 밝게 맥동
- 프레임 5-6: 로브가 오른쪽으로 살짝 흔들림
- 눈의 발광 강도가 수정 맥동과 동기화

- 프레임 5-6: 보라색 에너지 방출

## 7. 이펙트 디자인

### 영혼 속박 (Soul Bind)
| 항목 | 값 |
|------|---|
| **색상** | 보라 (`#6A146A` → `#9D2AA5`) |
| **형태** | 타겟에서 리치로 흐르는 체인/연결선 |
| **파티클** | 6-8개 작은 영혼 조각 (반투명) |
| **지속 시간** | 0.8초 |
| **해상도** | 32 x 64 픽셀 |

### 시체 부활 (Raise Dead)
| 항목 | 값 |
|------|---|
| **색상** | 녹색 (`#2D8F2D` → `#5FD35F`) |
| **형태** | 바닥에서 올라오는 육각형 마법진 |
| **파티클** | 위로 올라가는 작은 빛 입자 |
| **지속 시간** | 0.5초 |
| **해상도** | 64 x 64 픽셀 |

### 피격 플래시 (Hit Flash)
| 항목 | 값 |
|------|---|
| **색상** | 흰색 → 원래 색상 |
| **형태** | 외곽선 및 전체 밝기 증가 |
| **지속 시간** | 0.1초 |

### 눈 발광 (Eye Glow)
| 항목 | 값 |
|------|---|
| **색상** | 밝은 보라 (`#9D2AA5`) |
| **형태** | 눈구멍에서 바깥으로 퍼지는 빛 |
| **애니메이션** | 지속적 맥동 (Idle과 동기화) |

## 9. 시각적 대비 가이드

### 리치 vs 플레이어
| 요소 | 리치 | 플레이어 |
|------|------|----------|
| **색온도** | 차가운 보라/녹색 | 따뜻한 빨강/주황 |
| **동작** | 느리고 무거움 | 빠르고 민첩함 |
| **크기** | 화면 1/3 차지 | 화면 1/6 |
| **이펙트** | 어둡고 음산함 | 밝고 역동적 |

### 상태별 시각 표현
| 상태 | 변화 |
|------|------|
| **정상** | 기본 팔레트 |
| **버프** | 보라색 강도 +20%, 발광 강화 |
| **약화** | 채도 -30%, 파티클 감소 |
| **분노 (저체력)** | 빨간 악센트 추가 (`#8F2D2D`) |
