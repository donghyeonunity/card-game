## 1. 화면 개요

### 1.1 목적
매 런(run) 시작 시 플레이어의 캐릭터와 특성을 선택하는 화면

### 1.2 타겟 플랫폼

- **지원**: 1366x768 이상

### 1.3 디자인 스타일
- 다크 테마 판타지 게임 UI
- 기존 프로토타입 BattleScene과 일관된 비주얼

**슬라이더 디자인:**

| 요소 | 스펙 |
|------|------|

**임계값별 시각 효과:**

| 점수 | 색상 | 효과 |
|------|------|------|
| 0점 | 회색 (`#666`) | 없음 |
| 1점 | 특성색 연한 버전 (opacity 0.5) | 기본 카드 표시 |
| 3점 | 특성색 기본 | 강화 카드 표시 |
| 5점 | 황금색 (`#f4d03f`) | 최강 카드 + 패시브 + 반짝임 효과 |

**카드 미리보기:**
- 0점: "(없음)" 텍스트
- 1점+: 카드 아이콘 + 이름 표시
- 애니메이션: 카드 변경 시 페이드 + 슬라이드

**남은 포인트 표시:**
```
남은 포인트: ●●●○○ (3/5)
```
- 위치: 슬라이더 영역 하단
- 채워진 점: `#f4d03f` (황금색)
- 빈 점: `#444`

## 5. 애니메이션 스펙

### 5.1 Framer Motion 설정

**공통 전환:**
```typescript
const transition = {
  type: 'spring',
  stiffness: 300,
  damping: 25
};
```

**호버 효과:**
```typescript
whileHover={{ scale: 1.05, x: 10 }}
whileTap={{ scale: 0.95 }}
```

### 5.2 컴포넌트별 애니메이션

| 컴포넌트 | 트리거 | 애니메이션 |
|----------|--------|------------|
| CharacterCard | 선택 | 글로우 페이드 인 (0.3s) |
| TraitSlider | 임계값 도달 | 핸들 스케일 펄스 (1 → 1.2 → 1) |
| CardThumbnail | 카드 획득 | 슬라이드 인 (왼쪽에서 오른쪽) |
| ComboBonusBar | 조합 달성 | 슬라이드 업 + 글로우 확산 |
| StartButton | 활성화 | 버튼 바운스 |

### 5.3 페이지 전환

**진입:**
```typescript
initial={{ opacity: 0 }}
animate={{ opacity: 1 }}
transition={{ duration: 0.5 }}
```

**게임 시작:**
```typescript
exit={{ scale: 0.9, opacity: 0 }}
transition={{ duration: 0.3 }}
```
