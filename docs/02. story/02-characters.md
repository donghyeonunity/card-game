# 플레이어 캐릭터

> **설계 원칙**:
> - 혼자 던전에 들어가는 것이 자연스러운 캐릭터
> - 전투에 자신감이 있는 캐릭터 (던전의 위험성을 알면서도 들어감)
> - 각자 다른 감정축을 대표

---

## 캐릭터 요약

| 캐릭터 | 컨셉 | 동기 | 감정축 |
|--------|------|------|--------|
| **법사** | 지식을 갈구하는 마법사 | 금지된 지식 추구 | 지식/야망 |
| **전사** | 전직 전사, 현재 농부 | 어머니의 병 치료 | 사랑/헌신 |
| **도적** | 전사의 옛 전우 | 친구를 찾으러 (할머니 부탁) | 우정/의리 |

---

## 법사

### 기본 정보

| 항목 | 내용 |
|------|------|
| **컨셉** | 지식을 갈구하는 마법사 |
| **동기** | 던전에 잠든 금지된/잊혀진 지식을 얻으려 함 |
| **전투 수단** | 마법 |

### 혼자인 이유

- 위험한 연구로 학계/마법사 길드에서 추방됨
- 추구하는 지식이 너무 위험해서 아무도 함께 가려 하지 않음
- (또는) 발견을 나누고 싶지 않은 비밀주의자

### 감정선

| 시점 | 감정 | 내면 |
|------|------|------|
| **시작** | 기대, 집착 | "드디어... 그 지식을 얻을 수 있다" |
| **진행** | 경이로움 | "예상보다 더 대단한 곳이군" |
| **🔥 중간** | 지식의 무게 | "이 지식은... 알아도 되는 것인가?" |
| **⭐ 최종** | 깨달음 | "알고 싶었던 것의 진정한 의미는..." |

### 핵심 대사 (예시)

> "지식에는 가치가 없다, 오직 이해만이 있을 뿐."
> "위험하다고? 무지함이야말로 진정한 위험이다."

## 전사

### 기본 정보

| 항목 | 내용 |
|------|------|
| **컨셉** | 전직 전사, 현재 농부 |
| **동기** | 어머니의 병 치료 - 치료법이 던전에 있다는 소문 |
| **전투 수단** | 검/창 (과거 전쟁 경험) |

### 배경

```
과거: 전쟁터에서 싸운 전사
  ↓
현재: 전쟁이 끝나고 고향 마을로 돌아와 농사
  ↓
사건: 어머니가 원인불명의 병에 걸림
  ↓
결심: 던전에 치료법이 있다는 소문을 듣고 떠남
```

### 혼자인 이유

- 시골 외딴 마을, 용병 고용할 돈 없음
- 마을 사람들은 던전을 두려워함
- 형제자매 없음 (또는 어린 동생은 어머니 곁을 지켜야 함)
- "갈 수 있는 사람은 나뿐이다"

### 감정선

| 시점 | 감정 | 내면 |
|------|------|------|
| **시작** | 결연함 | "전쟁에서 살아남았다. 던전도 못 이길 이유 없다" |
| **진행** | 그리움 | "어머니... 조금만 기다려주세요" |
| **🔥 중간** | 절망과 희망 사이 | "정말 치료법이 있는 걸까?" |
| **⭐ 최종** | 귀환의 다짐 | "드디어... 이제 돌아갈 수 있다" |

### 핵심 대사 (예시)

> "전쟁보다 무서울까. 어머니 살리러 가는 길인데."
> "꼭 돌아가겠습니다. 조금만 기다려주세요."

### 어머니에게 남긴 편지

```
어머니,

죄송합니다. 말씀도 못 드리고 떠납니다.
던전에 다녀오겠습니다.
소문이 사실이라면, 어머니를 고칠 수 있을 겁니다.

꼭 돌아오겠습니다.
조금만 기다려주세요.

                                       - 아들 올림
```

## 도적

### 기본 정보

| 항목 | 내용 |
|------|------|
| **컨셉** | 전사의 옛 전우 |
| **동기** | 친구(전사)를 찾으러 - 할머니(전사의 어머니)의 부탁 |
| **전투 수단** | 단검, 생존 기술 |

### 배경

```
과거: 전쟁터에서 전사와 함께 싸운 전우
  ↓
전쟁 후: 도적/용병으로 떠돌이 생활
  ↓
어느 날: 옛 친구(전사)를 보러 고향 마을 방문
  ↓
도착: 집에 친구는 없다. 병든 할머니와 편지만.
  ↓
결심: 할머니의 부탁 - "내 아들을 찾아줘..."
  ↓
출발: 던전으로 향한다
```

### 혼자인 이유

- 급히 친구를 찾으러 떠남
- 떠돌이 생활로 동료가 없음
- 친구 문제는 자기가 해결해야

### 감정선

| 시점 | 감정 | 내면 |
|------|------|------|
| **시작** | 걱정 + 짜증 | "혼자 가다니, 바보 같은 놈" |
| **진행** | 초조함 | "빨리 찾아야 해" |
| **🔥 중간** | 친구의 흔적 발견 | "여기 있었구나... 아직 살아있겠지?" |
| **⭐ 최종** | 재회 (또는...) | 엔딩에 따라 |

### 핵심 대사 (예시)

> "그 녀석, 혼자 무슨 짓이야. 내가 찾아야지."
> "할머니, 걱정 마세요. 제가 꼭 데려올게요."

## 캐릭터 연결 관계

```
┌─────────────┐         전쟁터 전우         ┌─────────────┐
│    전사     │◄─────────────────────────►│    도적     │
└─────────────┘                            └─────────────┘
       │                                          │
       │ 던전 먼저 진입                           │ 나중에 진입
       ▼                                          ▼
  ┌─────────────────────────────────────────────────────┐
  │                      던  전                          │
  │                                                      │
  │    [전사의 흔적]  ←  도적 플레이 시 발견 가능        │
  │                                                      │
  └─────────────────────────────────────────────────────┘
```

### 연결의 효과

| 포인트 | 효과 |
|--------|------|
| 두 캐릭터가 같은 세계에 존재 | 세계가 "살아있는" 느낌 |
| 편지 | 전사가 왜 떠났는지 도적 시점에서 알게 됨 |
| 할머니의 부탁 | 도적에게도 "지켜야 할 것" 부여 |
| 시간차 진입 | 도적이 전사의 흔적을 발견하는 연출 가능 |

## 플레이 경험 차이

| 캐릭터 | 플레이 느낌 | 핵심 질문 |
|--------|-------------|-----------|
| **법사** | "알고 싶다" | 지식의 끝에는 무엇이 있는가? |
| **전사** | "지키고 싶다" | 사랑하는 사람을 위해 어디까지 갈 수 있는가? |
| **도적** | "찾고 싶다" | 우정은 어디까지 이어지는가? |
