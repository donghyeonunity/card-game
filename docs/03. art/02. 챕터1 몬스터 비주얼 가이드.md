# 1챕터 몬스터 아트 가이드

## 📋 프로젝트 개요

**목적**: 리치의 성 1챕터 전투에 사용할 몬스터 픽셀 아트 스프라이트 3종 제작
**스타일**: 16비트 다크 판타지 픽셀 아트
**용도**: 프로토타입 전투 시스템
**파일 형식**: PNG (투명 배경)

---

## 🎨 아트 디렉션

### 전체 컨셉
- **세계관**: 리치가 지배하는 어둠의 성
- **분위기**: 우울하고 퇴폐적인 다크 판타지
- **테마**: 죽음, 부패, 타락한 기사도
- **색감**: 낮은 채도, 차가운 색조, 보라-회색 중심

### 시각적 일관성
- 모든 몬스터는 동일한 색상 팔레트 사용
- 2px 검정색 아웃라인 (#1A0F1F) 필수
- 픽셀 아트 특유의 명확한 윤곽선과 절제된 디테일
- 배경색(#1A0F1F ~ #4A3B52)과 명확히 대비되는 색상 사용

## 👾 몬스터 1: 방황하는 망자 (Wandering Dead)

### 기본 스펙
- **해상도**: 48 x 64 픽셀
- **난이도**: ★☆☆ (약함)
- **파일명**: `enemy_wandering_dead_idle.png`

### 외형 설명
생전 평범한 농민이었던 언데드. 죽음을 받아들이지 못한 영혼이 육신에 머물러 있는 상태.

**주요 특징**:
- 해진 평민복 (갈색/회색 계열, 시엔나 + 다크 브라운)
- 창백한 피부 (스톤 그레이 → 애시 화이트 그라데이션)
- 비틀거리는 자세 (몸이 약간 기울어짐, 불안정한 느낌)
- 맨손 (무기 없음, 손은 축 늘어뜨림)
- 눈에 희미한 슬픔 (작은 보라색 빛 - 딥 바이올렛)

**디테일 가이드**:
1. **머리/얼굴**:
   - 창백한 회색 피부
   - 텅 빈 눈에 희미한 보라빛 (#6A146A)
   - 헝클어진 머리카락 (다크 브라운)

2. **상체**:
   - 찢어지고 더러운 튜닉 (#5C3317)
   - 천이 너덜거리는 표현 (픽셀 누락)
   - 말라비틀어진 체형

3. **하체**:
   - 낡은 바지 (#4A3B52)
   - 맨발 또는 해진 신발

4. **자세**:
   - 허리 약간 구부림
   - 팔은 아래로 축 늘어뜨림
   - 무릎 살짝 굽힘
   - 전체적으로 힘없는 느낌

## ⚔️ 몬스터 2: 타락한 검사 (Corrupted Swordsman)

### 기본 스펙
- **해상도**: 48 x 64 픽셀
- **난이도**: ★★☆ (중간)
- **파일명**: `enemy_corrupted_swordsman_idle.png`

### 외형 설명
생전 숙련된 전사였으나 리치의 힘에 타락한 언데드 검사. 검술의 기억은 남아있지만 영혼은 사라진 상태.

**주요 특징**:
- 녹슬고 금 간 갑옷 (러스트 골드 + 스톤 그레이)
- 투구 착용, 틈새로 텅 빈 눈빛 (보라색 빛 - 로열 퍼플)
- 한 손에 검을 들고 있음
- 정석적이지만 뒤틀린 전투 자세
- 검사다운 위엄이 아직 남아있음

**디테일 가이드**:
1. **투구**:
   - 녹슨 금속 (#6B5D3A)
   - T자형 안면부 틈새
   - 눈 부분에서 보라색 빛 (#9D2AA5)

2. **갑옷**:
   - 체인메일 또는 경갑 (#4A3B52 베이스)
   - 녹과 금 표현 (러스트 골드 얼룩)
   - 어깨 갑주

3. **무기**:
   - 한손검 (롱소드)
   - 칼날은 회색 (#4A3B52)
   - 손잡이는 어두운 색

4. **자세**:
   - 정석 전투 자세 (중단 구도)
   - 검을 45도 각도로 들고 있음
   - 방패는 없음
   - 안정적이지만 생기 없는 자세

## 🛡️ 몬스터 3: 리치의 기사 (Lich Knight) - 엘리트

### 기본 스펙
- **해상도**: 64 x 96 픽셀 (일반 적보다 큼)
- **난이도**: ★★★ (강함, 엘리트)
- **파일명**: `enemy_lich_knight_idle.png`

### 외형 설명
리치 보스가 직접 창조한 최정예 언데드 기사. 강력한 마법으로 강화된 갑옷과 무기를 보유.

**주요 특징**:
- 검은 판금 갑옷 (어두운 보라/검정 - 다크 퍼플 + 보이드 블랙)
- 붉은 망토 (악센트 컬러 - 다크 레드)
- 투구 안에서 푸른 불빛 타오름 (보라-파랑 - 로열 퍼플 + 하이라이트)
- 리치의 문장이 새겨진 대검
- 완벽한 전투 자세, 위압감
- 일반 적보다 크고 디테일한 디자인

**디테일 가이드**:
1. **투구**:
   - 뿔 달린 투구 (2개의 작은 뿔)
   - 눈 부분에서 강렬한 불꽃 (#9D2AA5 + #FFFFFF)

   - 어깨에서 허리까지
   - 바람에 살짝 날리는 느낌

4. **무기**:

   - 칼날 길이 길게 표현
   - 검에 보라색 룬 문자 새겨짐
   - 마법 오라 표현 (희미한 빛)

5. **자세**:
   - 완벽한 상단 자세
   - 검을 어깨 위로 들고 있음

## 🔧 제작 가이드라인

### 픽셀 아트 기본 원칙
1. **아웃라인**: 모든 형태에 2px 검정색 아웃라인 필수
2. **색상 제한**: 한 몬스터당 최대 12-16색 사용
3. **디테일 절제**: 작은 크기에서 알아볼 수 있도록 단순화
4. **대비**: 주요 부위는 명확한 색상 대비로 구분
5. **가독성**: 4배 확대 시 픽셀이 명확히 보여야 함

### 레이어 구조 (권장)
1. **배경**: 투명
2. **그림자**: 가장 어두운 색 (#1A0F1F, #3B1F47)
3. **베이스 색상**: 주요 색상
4. **하이라이트**: 밝은 색상 (#C4B8CC, #9D2AA5)
5. **아웃라인**: 2px 검정색 (#1A0F1F)
6. **특수 효과**: 눈 빛, 마법 오라 등

### 난이도 차별화
- **약함 (망자)**: 작고 허약한 실루엣, 단순한 디자인
- **중간 (검사)**: 균형잡힌 비율, 중간 디테일
- **강함 (기사)**: 크고 위압적, 높은 디테일, 특수 효과

## 🎮 사용 방법

### 1. HTML 생성기 사용
1. `/docs/enemy_sprite_generator.html` 파일을 브라우저에서 열기
2. 각 몬스터 카드에서 "템플릿" 버튼으로 기본 형태 생성
3. 팔레트에서 색상 선택 후 픽셀 단위로 그리기
4. "외곽선" 버튼으로 자동 아웃라인 추가
5. "다운로드" 버튼으로 PNG 파일 저장

### 2. 외부 AI 도구 사용 (선택사항)
- **DALL-E 3**: 위 프롬프트를 사용하여 이미지 생성
- **Stable Diffusion**: LoRA 모델 + 픽셀 아트 프롬프트
- **Midjourney**: `--style raw --ar 3:4` 옵션 사용

생성된 이미지를 48x64 (또는 64x96)로 리사이징 후 픽셀화 처리 필요.

### 3. 수동 제작 (권장)
- **Aseprite**: 전문 픽셀 아트 도구
- **Piskel**: 무료 온라인 픽셀 에디터
- **GraphicsGale**: 무료 픽셀 아트 소프트웨어
