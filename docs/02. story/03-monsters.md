# 몬스터 설정

> **핵심 컨셉**: 던전은 몬스터들의 세상. 플레이어는 침입자.

---

## 몬스터 세계관 요약

• 왜 존재하는지 스스로도 모름 (또는 본능적으로 알 뿐)
• 던전 밖에서는 존재할 수 없음 (추정)

"우리는 던전에서 태어났다. 던전이 곧 어머니."
```

---

## 2. 생존 방식

### 마력 + 약육강식 + 침입자 사냥

| 방식 | 설명 |
|------|------|
| **기본 생존: 던전 마력** | 던전이 뿜어내는 마력을 흡수하며 삶. 이것만으로도 "살 수는" 있음 |
| **성장: 약육강식** | 다른 몬스터를 먹으면 더 강해짐. 보스 = 수많은 동족을 먹고 정점에 오른 자 |
| **최고의 먹이: 침입자** | 인간의 생명력은 마력보다 훨씬 강력. 침입자를 먹으면 급격히 강해짐 |

### 생태계 흐름

```
        던전의 마력
            │
            ▼
    ┌───────────────┐
    │   일반 몬스터  │ ← 마력만으로 생존
    └───────────────┘
            │
            │ 잡아먹힘
            ▼
    ┌───────────────┐
    │  엘리트 몬스터 │ ← 동족을 먹고 성장
    └───────────────┘
            │
            │ 잡아먹힘
            ▼
    ┌───────────────┐
    │     보스      │ ← 수많은 희생 위에 군림
    └───────────────┘
            │
            │ 침입자 사냥
            ▼
    ┌───────────────┐
    │   더 강해짐   │ ← 인간은 최고의 먹이
    └───────────────┘
```

### "돌아온 자가 없다"의 의미

오래된 몬스터, 특히 보스급은 수많은 도전자들을 먹어왔다.
그들 안에는 이전 도전자들의 흔적이 있을 수 있다.

## 3. 목표/욕망

### 생존 본능

```
• 복잡한 목표 없음. 그저 살아남으려 함
• 먹고, 영역 지키고, 위협 제거
• 인간처럼 야망이나 계획이 없음
• 야생 동물과 비슷 - 순수한 본능

"왜 싸우는가? 살기 위해서다. 그것뿐."
```

### 스토리적 의미

| 포인트 | 효과 |
|--------|------|
| **악의 없는 적** | 몬스터는 "악당"이 아니라 "생존자" |
| **플레이어의 위치** | 우리도 살기 위해 싸운다, 그들도 마찬가지 |
| **도덕적 회색** | "누가 옳은가?" 없음. 그냥 생존 경쟁 |
| **자연의 섭리** | 야생의 법칙, 강한 자가 살아남음 |

## 4. 사회 구조

### 종족 무리 + 힘의 서열 + 약자 연대

| 구조 | 설명 |
|------|------|
| **종족별 무리** | 같은 종끼리 뭉침. 다른 종은 경쟁자/먹이 |
| **힘의 서열** | 무리 안에서 강한 자가 지배. 약한 자는 두려움으로 복종 |
| **약자 연대** | 가장 약한 개체들은 떼로 뭉침. 수적 우위로 생존 확률 높임 |

### 계층 구조

```
                    ┌─────────────┐
                    │    보스     │  종족의 정점
                    │  (1마리)    │  절대적 힘
                    └──────┬──────┘
                           │
              지배/두려움  │
                           ▼
                ┌─────────────────┐
                │    엘리트       │  보스 아래 강자들
                │   (소수)        │  영역 내 2인자들
                └────────┬────────┘
                         │
            통솔/착취    │
                         ▼
        ┌────────────────────────────────┐
        │         일반 몬스터            │  다수
        │  ┌─────┐ ┌─────┐ ┌─────┐       │  무리 지어 생존
        │  │ 떼  │ │ 떼  │ │ 떼  │       │  약할수록 뭉침
        │  └─────┘ └─────┘ └─────┘       │
        └────────────────────────────────┘
```

## 5. 게임플레이 연결

| 구조 | 게임에서 표현 |
|------|---------------|
| **종족별 무리** | 구역마다 다른 몬스터 종족 테마 |
| **힘의 서열** | 보스 = 그 구역 종족의 왕 |
| **약자 연대** | 일반몹은 무리로 등장, 엘리트는 단독/소수 |
| **떼 전투** | 약한 몬스터 여러 마리 vs 플레이어 |

### 몬스터 등급 (PRD 기준)

| 등급 | 역할 | 플레이어 감정 | 생태계 위치 |
|------|------|---------------|-------------|
| **일반** | 기본 전투, 리듬 형성 | 가벼운 긴장감 | 하위층, 떼로 생존 |
| **엘리트** | 도전적 전투 | 집중과 긴장 상승 | 중간층, 소수 강자 |
| **보스** | 구역 마무리 | 클라이맥스, 성취감 | 정점, 종족의 왕 |
